Projet Parkslot

Application web de gestion de parking avec attribution automatique des places, réservations et file d'attente.

Contexte du projet

Ce projet a été réalisé dans le cadre de ma formation BTS SIO. L'objectif était de développer une application web permettant d'attribuer automatiquement des places de parking aux membres du personnel, de gérer les réservations et la file d'attente lorsque toutes les places sont occupées.

Durée : 5 semaines

Équipe : Projet collaboratif

Objectifs du projet

  • Gérer les places de parking de manière automatisée
  • Permettre aux membres du personnel de réserver une place
  • Gérer une file d'attente lorsque toutes les places sont occupées
  • Administrer les abonnements et les utilisateurs
  • Assurer une interface moderne et responsive

Fonctionnalités principales

🚗

Attribution automatique

Attribution intelligente des places de parking selon la disponibilité et les préférences des utilisateurs.

📅

Réservation

Réservation de places pour une période donnée avec confirmation par email.

⏳

File d'attente

Gestion automatique de la file d'attente lorsque toutes les places sont réservées.

👑

Administration

Interface d'administration pour gérer les places, les utilisateurs et les abonnements.

🗓️ Organisation & Journal de bord

Semaine 1

Analyse & Conception

Analyse des besoins, maquettage sur Figma, conception de la base de données.

Semaine 2

Développement Backend

Mise en place de l'authentification, modèles Eloquent, contrôleurs et routes.

Semaine 3

Développement Frontend

Intégration des vues Blade, design responsive avec Tailwind CSS.

Semaine 4-5

Tests & Déploiement

Tests fonctionnels, correction des bugs, déploiement avec Docker.

Technologies utilisées

Laravel PHP MySQL Tailwind CSS JavaScript Vite Docker Git Figma

Ce que j'ai appris

  • Utilisation avancée du framework Laravel (MVC, Eloquent ORM, migrations)
  • Gestion des relations entre modèles (hasMany, belongsTo, belongsToMany)
  • Implémentation d'une logique de file d'attente automatisée
  • Développement avec Tailwind CSS pour un design responsive
  • Conteneurisation avec Docker pour faciliter le déploiement
  • Travail en équipe avec Git (branches, pull requests)
